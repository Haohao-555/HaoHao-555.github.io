我的小项目

项目一：云音乐全栈小程序 2020-07

后台系统

小程序端

欢迎扫码体验

项目描述

  • 小程序端-音乐菜单中的歌单列表
    • 通过API获取歌单列表
    • 点击歌单（歌曲），加载歌单信息（播放页面）
    • 点击进度条，播放歌词和播放效果随着联动
    • 歌词可播放、暂停、切换
  • 小程序端-发现菜单（类似博客功能）
    • 发布功能
    • 评论和分析功能
    • 模糊查询功能
  • 小程序端-我的菜单
    • 歌单播放历史
    • 我的发布
    • 小程序码
  • 后台管理-歌单管理、轮播图管理、博客管理

技术栈

  • 小程序端：云开发（云函数 + 云调用 + 云数据库 + HTTP API + 云存储）、组件化开发
  • vue-admin-template 搭建管理系统前端
  • koa2 搭建管理系统后端

项目二：vue全家桶高仿小米购物网站 2020-09

技术栈

  • 前端方面：VueCli4.0 脚手架、vue全家桶 [ vue（整体架构） + vuex（状态管理） + vue-router（路由） + axiso（请求） + element-ui（使用部分弹框）]
  • 数据方面：mockjs（模拟数据） + vue-cookie（cookie存储）
  • 其他：node-sass + node-loader + vue-lazyload（图片懒加载）

项目三：vue + koa2 仿微博 2021-02

项目描述

  • 首页：查看微博（自己 和 关注人）、粉丝、关注人
  • 我的空间：查看微博（自己 @ 我的微博）、粉丝、关注人
  • 广场：查看微博（所有人）
  • 设置页：修改个人信息、修改密码、退出登录
  • 其他人主页：查看微博（其他人）、关注或取消关注
  • 发布微博可以 @ 某人

技术栈

  • 前端：vue 全家桶 + element UI + bootstap
  • 后端：node + koa2 + koa-router + koa-redis + mysql + sequlize + pm2 + suertest（单元测试）
  • 数据方面：mysql-5.7 + redis
  • 其他：vue-at + textarea-caret + node-sass + sass-loader
  • 服务器：阿里ECS服务器
