我的第一篇博客

一、本地使用 hexo 搭建个人博客

  • 全局安装 hexo
    • npm install -g hexo-cli
    • hexo -v (查看 hexo 信息)

  • 创建文件 blog

    • 进入到文件中并初始化项目 hexo s

  • 启动项目

    • hexo s

二、创建博客

  • hexo n “我的第一篇博客”

  • 重新启动项目

    • hexo s

三、部署上线

  • github 创建一个仓库

    • 仓库名就是访问的域名

    • 修改本地项目的 _config.yml 文件

  • 在项目目录下安装 hexo-deployer-git

    • npm install – save hexo-deployer-git

  • 打开本地终端生成文件，再将项目 push 到 github

    • hexo g (生成文件)
    • hexo d (push 到远端)

  • 通过 Haohao-555.github.io 访问
