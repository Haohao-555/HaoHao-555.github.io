    # 一、介绍

    • 涉及到的身份为学生，管理员，（教师身份由于时间不足并没有涉及到）
    • 学生功能模块：关注，退选课程，绑定及查看个人信息，注册账号
    • 管理员功能模块：查看学生详细信息（不能进行修改），对课程及教师信息进行管理

    # 二、技术

    • 后端：servlet+tomcat+Mysql+navicat数据库可视化+myeclipse
    • 前端：jsp+html+ajax+Bootstrap+jquery
